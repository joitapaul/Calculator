

## 📌 Project Overview

This project is a simple **Calculator application** that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
It is designed to demonstrate the implementation of fundamental programming concepts and user interaction.

## 🚀 Features

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Clear/Reset functionality
* Simple and user-friendly interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
Calculator/
│
├── index.html       # Main HTML file
├── style.css        # Styling for calculator UI
├── script.js        # Calculator logic
└── README.md        # Project documentation
```

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/calculator.git
```

2. Open the project folder.

3. Run the **index.html** file in any web browser.
